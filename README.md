# ❤️ Corazón I Love You

Proyecto desarrollado con **Python** que dibuja un corazón utilizando la frase **"I Love You"**. La animación forma el contorno del corazón, posteriormente lo rellena completamente con la misma frase y, al finalizar, muestra un mensaje especial en color blanco.

---

## 🚀 Tecnologías

- Python

---

## ✨ Características

- ❤️ Formación del corazón palabra por palabra
- 💖 Relleno completo utilizando "I Love You"
- ✨ Animación fluida y atractiva
- 🤍 Mensaje final resaltado en color blanco
- 🎨 Proyecto de programación creativa

---

## ▶️ Cómo ejecutar

1. Clona este repositorio.
2. Ejecuta el proyecto:

```bash
python heart_love_you.py
```
